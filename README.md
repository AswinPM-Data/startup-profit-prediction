# Startup Profit Prediction using Multiple Linear Regression

## Project Overview
Analyzed how R&D, Administration, and Marketing spending
influence the profitability of startups using Multiple Linear Regression.

## Dataset
- 50 startups across New York, California, and Florida
- Features: R&D Spend, Administration Spend, Marketing Spend, State
- Target: Net Profit

## Key Findings
- R&D Spend is the strongest driver of profit
- Administration Spend has very little impact on profit
- Marketing Spend has moderate but inconsistent influence
- Average profit: New York ~$128K | California ~$126K | Florida ~$125K

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Multiple Linear Regression
- Power BI (KPI Dashboard)
- Excel (Data Exploration)

## Business Recommendations
- Prioritize R&D investment for maximum profit growth
- Reduce unnecessary administration costs
- Balance marketing spend with R&D for stable returns
